#### 使用说明

可以自行向目录下的data文件夹中添加json文件作为自定义题库，格式必须严格按照软件自带的json文件。

每道题只可提交一次，右上角可以切换题库，点击错题模式按钮后即可重做此次启动软件后所做错的题。

需要.NET Framework 4.6运行环境，安装后根据提示到官网下载安装即可。

祝考试顺利。
